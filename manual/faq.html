    How force ZetaPush SDK to use https instead http ?

    ZetaPush client allow you to declare a specific apiUrl

    const client = new ZetaPush.Client({
  apiUrl: 'https://api.zpush.io/',
  businessId: '<YOUR-BUSINESS-ID>'
})
